# [2주차] 트랙터 동력 시스템 및 경제성 분석 (TCO)

## 1. 실습 개요
본 실습에서는 트랙터의 두 가지 주요 동력원인 **내연기관(디젤)**과 **전동화(배터리)** 시스템의 경제성을 정량적으로 비교합니다. 초기 구매가와 가변 비용(연료비, 유지보수비)을 기반으로 10년 총소유비용(TCO)을 계산하고 최적의 투자 시점을 분석합니다.

---

## 2. 주요 실습 내용
- 다양한 트랙터 제원 데이터 입력 및 TCO 계산 엔진 구현
- 연간 작업 시간 및 에너지 가격 변화에 따른 민감도 분석
- Matplotlib을 이용한 비용 역전 지점(Break-even Point) 시각화

---

## 3. 핵심 알고리즘: TCO 계산식
$$TCO = P_{initial} + \sum_{t=1}^{n} (Cost_{energy, t} + Cost_{maint, t})$$
- $P_{initial}$: 초기 구매 가격
- $Cost_{energy}$: 연간 에너지 비용 (연료 소모율 × 가동 시간 × 단가)
- $Cost_{maint}$: 연간 소모품 및 정비 비용

---

## 4. 실습 코드 실행 안내

### [Step 1] 파이썬을 이용한 TCO 시뮬레이션
제공된 `step1_tco_analysis.py` 파일을 실행하여 디젤 트랙터와 전기 트랙터의 누적 비용 곡선을 확인합니다.

```bash
# 관련 라이브러리 설치
pip install matplotlib numpy

# 코드 실행
python step1_tco_analysis.py
```

### [Step 2] 시나리오 분석 (Self-Task)
코드 내의 매개변수를 다음과 같이 수정해 보며 결과 변화를 관찰하십시오.
1. 연간 가동 시간을 500시간으로 줄였을 때의 경제성 변화
2. 디젤 가격이 2,500원/L로 급등했을 때의 손익분기점 단축 효과
3. 정부 보조금(전기차 보조금 제도를 가정)을 구매가에 반영했을 때의 효과

---

## 5. 실습 결과 해석 가이드
- **초기 비용**: 전기 트랙터는 배터리 가격으로 인해 디젤 대비 약 40~50% 비쌈
- **운영 비용**: 전기 트랙터의 연간 운영비는 디젤의 약 10~20% 수준
- **결론**: 일정 수준 이상의 연간 가동 시간이 확보될 때 전동화 트랙터의 경제성이 확보됨
